# Logo Image

## Instructions

Place your logo image in this folder with the name: **logo.png**

### Recommended Specifications:
- **File Name**: `logo.png`
- **Format**: PNG (with transparent background preferred)
- **Dimensions**: 300x120 pixels (or similar 2.5:1 ratio)
- **File Size**: Under 100KB for optimal loading
- **Background**: Transparent or white

### Alternative Formats:
If you don't have a PNG, you can also use:
- `logo.jpg`
- `logo.svg` (best for scalability)
- `logo.webp`

### Fallback:
If no logo image is found, the navbar will display "Marcova" as text instead.

---

## Current Setup:
The logo will be displayed in the navbar at the top of the website with:
- Height: 48px (3rem)
- Width: Auto-adjusted to maintain aspect ratio
- Hover effect: Slight scale-up animation
